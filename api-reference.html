API Reference

This section provides detailed information about the AI Chatboard API endpoints and usage.

Endpoint

POST https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent

Request Format

{
  "contents": [{
    "parts": [{
      "text": "Your question here"
    }]
  }],
  "generationConfig": {
    "temperature": 0.7,
    "topK": 40,
    "topP": 0.95,
    "maxOutputTokens": 1024
  }
}

Response Format

The API returns a JSON object containing the generated content in the candidates array.

{
  "candidates": [
    {
      "content": {
        "parts": [
          {
            "text": "Generated response text here"
          }
        ]
      }
    }
  ]
}

Error Handling

  • 401 Unauthorized: Invalid API key.
  • 403 Forbidden: API key not authorized.
  • 404 Not Found: Endpoint not found.
  • 429 Too Many Requests: Rate limit exceeded.

Best Practices

  • Keep your API key secure and do not expose it publicly.
  • Handle errors gracefully in your application.
  • Respect rate limits to avoid service interruptions.
