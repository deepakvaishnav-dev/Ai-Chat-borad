Getting Started

This guide will help you quickly set up and integrate AI Chatboard into your project.

Step 1: Obtain API Key

Sign up for Google Cloud and enable the Gemini API. Generate your API key to use with AI Chatboard.

Step 2: Include AI Chatboard in Your Project

Add the necessary HTML, CSS, and JavaScript files to your project. You can use the provided files or customize as needed.

Step 3: Configure API Key

Replace the placeholder API key in script.js with your actual Gemini API key:

const API_KEY = 'YOUR_ACTUAL_API_KEY_HERE';

Step 4: Customize UI

Modify the styles in styles.css to match your branding and preferences.

Step 5: Test the Chat

Open the website and try chatting with the AI assistant. Ensure responses are received and displayed correctly.
