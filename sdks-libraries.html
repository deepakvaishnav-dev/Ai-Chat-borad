SDKs & Libraries

AI Chatboard provides official SDKs and libraries to simplify integration and development.

Available SDKs

  • JavaScript SDK: Easily integrate AI Chatboard into web applications.
  • Python SDK: Use AI Chatboard in backend services and scripts.
  • Other Languages: SDKs for popular languages coming soon.

Installation

Install the JavaScript SDK via npm:

npm install aichatboard-sdk

Install the Python SDK via pip:

pip install aichatboard-sdk

Usage Example (JavaScript)

import AIChatboard from 'aichatboard-sdk';

const chat = new AIChatboard({ apiKey: 'YOUR_API_KEY' });

chat.sendMessage('Hello!').then(response => {
    console.log(response);
});

Usage Example (Python)

from aichatboard import AIChatboard

chat = AIChatboard(api_key='YOUR_API_KEY')
response = chat.send_message('Hello!')
print(response)
